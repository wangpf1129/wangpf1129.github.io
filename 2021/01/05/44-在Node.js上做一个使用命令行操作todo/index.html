在Node.js上做一个使用命令行操作todo

源代码链接：https://github.com/wwwpppfffzzz/pf-node-todo

下载

下载npm包：

npm install -g pf-node-todo
// 或者
yarn global add pf-node-todo

下载后可查看版本：

todo --version

如何使用

  1. todo 查看所有任务列表
    • 能够操作所有增删改查
  2. todo add 添加一个任务
  3. todo clear 清空所有任务

单元测试

用来测试 读文件 和 写文件

yarn test 
// 或者
npm run test
